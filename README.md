## Apresenta.ai
Projeto para o componente curricular de Programação Orientada a Objetos

## Stack
### Back
- Java 17+
- Javalin (servidor HTTP REST)
- SQLite
- JUnit 5 (testes unitários)

### Front
- Electron
- React (componentes)
- Editor de markdown (Monaco)
- Biblioteca de UI (shadcn)

### Integrações
- Anthropic API (Claude) — provedor real principal
- Provedor Mock — usado em testes automatizados
